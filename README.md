# Student Information Portal (Roll No: 1234)

Full-stack assignment project using React SPA + Express API + MySQL.

## Run
1. Copy `backend/.env.example` to `backend/.env` and set MySQL credentials.
2. Import `sql/schema_1234.sql` in MySQL.
3. Run `npm run install:all`
4. Run `npm run start:prod`
5. Open `http://localhost:4000`
